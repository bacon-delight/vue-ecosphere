Here's a list of types that various components use

#### Global Types

###### Theme

```ts
type theme = "auto" | "light" | "dark" | "invert";
```

###### Hue

```ts
type hue =
	| "primary"
	| "primary-variant"
	| "secondary"
	| "secondary-variant"
	| "error"
	| "success"
	| "warning"
	| "information";
```
